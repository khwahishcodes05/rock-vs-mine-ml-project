# Sonar Rock vs Mine ML Project

This project uses Machine Learning to classify sonar signals as either rock or mine using Logistic Regression and data preprocessing techniques.

## 📊 Model Used
- Logistic Regression

## 📈 Accuracy
- ~83%

## ⚙️ Steps Followed
- Data loading and preprocessing
- Label encoding
- Feature scaling (StandardScaler)
- Train-test split
- Model training and prediction
- Evaluation using accuracy and classification report

## 📁 Dataset
Sonar dataset (UCI repository)

## 🚀 Goal
To predict whether an object detected by sonar is a rock or a mine.
